# Human Activity Recognition (HAR): Benchmarking Sequence Models with Attention Mechanisms

## 📌 Project Overview
This project focuses on identifying human activities (Walking, Sitting, Standing, etc.) from tri-axial accelerometer and gyroscope data. The core research objective was to design a robust sequence model capable of capturing long-range temporal dependencies in high-frequency sensor streams.

## 🚀 Technical Highlights
- [cite_start]**Architectural Benchmarking:** Conducted a rigorous comparison between 1D-CNNs, Simple RNNs, GRUs, and LSTMs.
- [cite_start]**Complex LSTM with Attention:** Developed a custom LSTM architecture integrated with a **Self-Attention Mechanism** to prioritize critical temporal segments in the sensor data[cite: 65].
- [cite_start]**State-of-the-art Performance:** Achieved a peak **accuracy of 91.5%**, significantly outperforming standard CNN and GRU baselines in robustness[cite: 65].
- [cite_start]**Explainable AI (XAI):** Implemented **Attention Heatmaps** to visualize which time steps contributed most to the activity classification, ensuring transparency in model decision-making.
